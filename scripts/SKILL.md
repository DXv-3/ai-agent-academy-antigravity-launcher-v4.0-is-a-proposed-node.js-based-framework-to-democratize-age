# Skill: Agent Builder (scripts/agent.js)

## Purpose
Generate a self-contained, runnable Node.js agent script from a natural language description. No external dependencies required — runs in template fallback mode out of the box. Full AI-powered generation is available when the `ai-agent-academy-antigravity-v4.0` package is installed.

---

## Invocation

```bash
node scripts/agent.js "<agent description>"
```

**Examples:**
```bash
node scripts/agent.js "research agent that summarizes arXiv papers"
node scripts/agent.js "monitor GitHub for new AI repos"
node scripts/agent.js "analyze CSV data and generate report"
node scripts/agent.js "generate weekly email digest"
```

Running without arguments prints usage instructions and exits cleanly.

---

## What It Does (Step by Step)

1. Reads the agent description from `process.argv` (all args joined as one string)
2. Pattern-matches the description against 4 task categories
3. Generates a boilerplate `.js` agent file with task-specific log hints
4. Writes the file to `agents/agent-<slugified-name>.js`
5. Prints the output path and a ready-to-run command

---

## Task Pattern Categories

| Pattern | Triggers On | Injected Hint |
|---|---|---|
| Research | `research`, `arxiv`, `paper` | Add fetch/axios to pull papers |
| Monitor | `monitor`, `track`, `watch` | Add cron or interval polling |
| Analyze | `analyze`, `analysis` | Add data processing logic |
| Generate | `generate`, `create`, `build` | Add file/output creation |
| (default) | anything else | General task mode |

---

## Output File Structure

Every generated agent follows this pattern:

```js
#!/usr/bin/env node
// Auto-generated agent: <your description>
// Generated by AI Agent Academy v4.0 (template mode)

const task = process.argv[2] || '<your description>';

async function run() {
  console.log('🤖 Agent starting task:', task);
  console.log('📝 Description:', '<your description>');
  // injected hint logic here
  console.log('✅ Agent task complete');
}

run().catch(console.error);
```

Output goes to: `agents/agent-<slug>.js`  
The slug is the description lowercased, non-alphanumeric chars replaced with `-`, truncated to 40 chars.

## Upgrade: Full AI Generation

To unlock LLM-powered agent generation (Gemini / Ollama):

```bash
cd ai-agent-academy-antigravity-v4.0 && npm install
```

Then use `src/agent-builder.js` directly or via the npm script:
```bash
npm run agent -- "your agent description"
```

The full builder uses `generateAgentCode()` which sends your prompt to an LLM and returns production-ready, task-specific Node.js code — not just template stubs.

## npm Script Shortcut

Defined in root `package.json`:
```json
"agent": "node scripts/agent.js"
```

So you can also run:
```bash
npm run agent -- "monitor Hacker News for AI stories"
```

---

## Key Functions

| Function | Role |
|---|---|
| `buildAgent(prompt)` | Entry point — orchestrates the full build flow |
| `templateAgent(prompt)` | Renders the agent file string from a prompt |
| `generateAgentLogic(prompt)` | Pattern-matches prompt → returns inline JS log/hint code |
| `ensureDir(dir)` | Creates `agents/` output directory if it doesn't exist |
| `log(msg, icon)` | Prefixed console.log helper |

---

## Extending This Skill

To add a new task pattern, push to the `patterns` array in `generateAgentLogic()`:

```js
{ test: /scrape|crawl|spider/i, log: '🕷️ Scraper task detected', hint: 'Add puppeteer or cheerio' },
```

To change the output directory, update `DIST_DIR`:
```js
const DIST_DIR = path.join(process.cwd(), 'my-agents');
```
