# SOUL.md — The Spirit of AI Agent Academy

> *"Democratize agentic AI. Make the impossible accessible."*

---

## Mission

AI Agent Academy exists to put the power of autonomous AI agents in the hands of everyone — not just engineers, not just researchers, not just enterprises. **Anyone** with an idea should be able to build, deploy, and share an AI agent in under 90 seconds.

---

## Core Values

### 1. 🌍 Radical Accessibility
Every feature must be usable by someone with zero coding experience. If a non-engineer can't use it, we haven't finished building it.

- No-code first: chat → agent → deployed
- Sensible defaults everywhere
- Error messages that explain *what to do*, not just *what went wrong*

### 2. 🚀 Speed Over Perfection
A working agent deployed in 90 seconds beats a perfect agent that takes 3 days. Ship fast, iterate faster.

- Templates over blank slates
- Autodetect over configuration
- One command over ten steps

### 3. 🤝 Open Ecosystem
Agents are more powerful when they work together. The marketplace isn't a walled garden — it's a commons.

- Publish freely, fork freely
- MCP bridges connect everything
- Swarms multiply individual capability

### 4. 🔒 Safety by Default
Autonomy without guardrails is chaos. Every agent runs through policy hooks before executing.

- Terminal command allowlists
- Browser action approval flows
- Deploy pre-flight checks
- No silent failures

### 5. 🧠 Model Agnostic
We don't pick winners. Gemini 3 Pro, Ollama, GPT-4, Claude — use what works for you.

- Autodetect available models
- Graceful fallbacks
- Template-only mode when no model is available

## The Antigravity Vision

Google's Antigravity IDE represents a paradigm shift: from *writing code* to *directing agents*. AI Agent Academy is the on-ramp to that future.

We believe:
- **The best IDE is one that understands your intent**, not just your syntax
- **Agents should be first-class citizens** in every development workflow
- **The gap between idea and deployed software** should be measured in seconds, not weeks

## What We Are Not

- ❌ Not a replacement for deep engineering — we're an accelerator
- ❌ Not locked to any single cloud provider
- ❌ Not a black box — every agent is inspectable, forkable, improvable
- ❌ Not finished — this is v4.0 of an ongoing journey

## The 90-Second Promise

From zero to deployed agent in 90 seconds:

```
t=0s   → npm run no-code
t=5s   → Type: "research agent that summarizes AI papers"
t=10s  → Model detected, template selected
t=30s  → Agent code generated
t=60s  → Agent tested locally
t=90s  → Agent deployed to Antigravity workspace ✅
```

This is the standard we hold ourselves to.

---

## For Future Contributors

When you add a feature, ask:
1. Can a non-engineer use this without reading docs?
2. Does it work offline (Ollama fallback)?
3. Does it respect the safety hooks?
4. Does it make the 90-second promise easier to keep?

If yes to all four: ship it.
